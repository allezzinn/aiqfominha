# aiqfominha

Esse projeto consta com três tabelas: usuarios, pedidos e itens_pedido.
É possivel cadastra usuários com senhas criptografadas que acessar as aplicações do sistema a partir de token de autentificação.
Há Apis com os métodos: GET, PUT, POST e DELETE
Apis como: login, listar, adicionar item ao pedido, etc..
